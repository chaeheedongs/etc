# basic syntax

<br>
<br>
<br>

## header

* This is Header

```
# H1
## H2
### H3
```

# H1
## H2
### H3

<br>
<br>
<br>

## bold

* This is Bold

```
**bold text**
```

**bold text**

<br>
<br>
<br>

## Italicized text

* This is italicized text

```
*italicized text*
```

*italicized text*

<br>
<br>
<br>

## blockquote

* This is blockquote

```
> blockquote
```

> blockquote

<br>
<br>
<br>

## ordered list

* This is ordered list

```
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

<br>
<br>
<br>

## unordered list

* This is unordered list

```
- First item
- Second item
- Third item
```

- First item
- Second item
- Third item

<br>
<br>

## code

* This is code

```
`code`
```

`code`

<br>
<br>

```

* This is link
